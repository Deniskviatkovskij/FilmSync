Project FilmSync - watching movies and videos with friends made easier!

https://filmsync.net

FEATURES:

Lobby system.

Synchronized YouTube and Twitch videos/streams.

Live chat.

TicTacToe game.

Give host permissions and kick unwanted guests.

Change the color theme of the page.

Change the language.
